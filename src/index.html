Projects

CS61CPU

Keywords: Computer Architecture, CPU, Pipelined CPU, Logisim

CS61CPU is a 5-stage pipelined CPU using logisim. It supports 32-bit instructions and has a 32-bit data bus. The CPU has a 32-bit instruction memory and a 32-bit data memory. The CPU has a 5-stage pipeline: IF, ID, EX, MEM, and WB. It is optimized for performance and has a 2-cycle branch prediction. The CPU has a 32-bit ALU and supports 32-bit arithmetic operations.

NUMC

Keywords: Matrix Operations, SIMD, Multi-threading

NUMC is a python library that provides a set of functions to perform matrix operations. It supports matrix addition, matrix subtraction, matrix multiplication, matrix division, matrix transpose, matrix determinant, matrix inverse, and matrix rank. The library is optimized for performance by using SIMD instructions and multi-threading. The library is designed to be easy to use and efficient.
